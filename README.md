# Business Intelligence Project – ComptoirDB Data Warehouse

## Overview
This project demonstrates the implementation of a Business Intelligence system using a data warehouse and data mining 
techniques.
The objective is to analyze sales data from the ComptoirDB database in order to support decision making through 
dashboards and analytical models.

## Tools & Technologies
- SQL Server – database management
- Talend Open Studio – ETL process
- Power BI / Excel – dashboards and reporting
- Weka – data mining algorithms

## Project Architecture
The system follows a typical Business Intelligence architecture:

1. Operational Database (ComptoirDB)
2. ETL process using Talend
3. Data Warehouse with a Star Schema
4. Analytical dashboards
5. Data mining analysis

## Dashboards & Analysis
The dashboards allow analysis of:
- Sales by product category
- Quantity sold by product
- Sales evolution over time
- Customer purchasing behavior

## Data Mining
The K-Means clustering algorithm was applied using Weka to group customers according to delivery delays.
This analysis helps identify different customer behavior patterns.
